Quine-McCluskey Algorithm

The Quine-McCluskey (QMC) method is a tabular algorithm for Boolean function minimization. Unlike K-Maps, it scales to any number of variables and is suitable for computer implementation.

Developed by

Quine (1952), McCluskey (1956)

Variables

Any number (practical ~15)

Complexity

O(3ⁿ) worst case

Result

Guaranteed minimum

The Algorithm

#
Quine-McCluskey Steps
  1. List minterms in binary and group by number of 1s
  2. Compare adjacent groups — combine terms differing by one bit
  3. Mark combined terms with a dash (-) for the differing bit
  4. Repeat step 2-3 until no more combinations possible
  5. Unchecked terms are Prime Implicants (PIs)
  6. Build PI chart to find Essential Prime Implicants (EPIs)
  7. Select minimum cover of remaining minterms

Why it works: XY + XY' = X (absorption). QMC systematically finds all such combinations, then selects the minimum set.
