Sequential Counter

Synchronous Counter

All flip-flops share the same clock signal and change state simultaneously. No ripple delay — outputs are glitch-free. The standard choice for modern digital design.

Clock

Common to all FFs

Propagation

1 × tpd

Glitches

None

Logic

Carry chain needed

How It Works

#

Each flip-flop receives the same clock. The toggle enable (T input) for each stage is determined by combinational logic:

  • T0 = 1 (always toggles)
  • T1 = Q0 (toggle when Q0=1)
  • T2 = Q0 · Q1 (toggle when both lower bits are 1)
  • Tn = Q0 · Q1 · ... · Q(n-1)

Toggle Condition for bit n

Tn = Q0 · Q1 · ... · Qn-1

Carry Chain & Lookahead

#

The T input depends on an AND chain of all lower bits. This creates a carry propagation problem similar to adders:

Ripple Carry

Tn = Tn-1 · Qn-1

Simple but slow for wide counters

Carry Lookahead

Tn = Q0·Q1·...·Qn-1

Parallel AND — faster but more gates

Practical limit: For counters wider than ~16 bits, use hierarchical or pipelined designs to meet timing. Most synthesis tools handle this automatically when you write Q <= Q + 1.

Up/Down Counter

#

Add a direction control input. Toggle condition uses Q or Q' based on direction:

DIR Operation Tn condition
0Count UpQ0 · Q1 · ... · Qn-1
1Count DownQ'0 · Q'1 · ... · Q'n-1

Up/Down Toggle Logic

Tn = (DIR' · Qn-1 + DIR · Q'n-1) · Tn-1

Loadable (Presettable) Counter

#

Add a parallel load capability to initialize the counter to any value. Essential for:

  • MOD-N counters: Load (2N - MOD) to count up to overflow
  • Timers: Load countdown value
  • Address generators: Jump to specific address
LOAD EN Operation
1XQ ← D (parallel load)
01Q ← Q + 1 (count)
00Q ← Q (hold)

RTL (Verilog / VHDL)

#
Verilog sync_counter.v 
// Simple N-bit Synchronous Up Counter
module sync_counter #(
    parameter WIDTH = 8
)(
    input  wire clk,
    input  wire rst_n,
    input  wire en,
    output reg [WIDTH-1:0] Q
);

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            Q <= {WIDTH{1'b0}};
        else if (en)
            Q <= Q + 1'b1;
    end

endmodule

// Up/Down Counter with Load
module updown_counter #(
    parameter WIDTH = 8
)(
    input  wire clk,
    input  wire rst_n,
    input  wire en,
    input  wire dir,       // 0=up, 1=down
    input  wire load,
    input  wire [WIDTH-1:0] D,
    output reg [WIDTH-1:0] Q,
    output wire tc         // Terminal count
);

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            Q <= {WIDTH{1'b0}};
        else if (load)
            Q <= D;
        else if (en) begin
            if (dir)
                Q <= Q - 1'b1;  // Down
            else
                Q <= Q + 1'b1;  // Up
        end
    end

    // Terminal count: all 1s (up) or all 0s (down)
    assign tc = dir ? (Q == {WIDTH{1'b0}}) : (Q == {WIDTH{1'b1}});

endmodule

// MOD-N Counter (e.g., MOD-10 decade counter)
module mod_n_counter #(
    parameter MOD = 10,
    parameter WIDTH = $clog2(MOD)
)(
    input  wire clk,
    input  wire rst_n,
    input  wire en,
    output reg [WIDTH-1:0] Q,
    output wire tc
);

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            Q <= {WIDTH{1'b0}};
        else if (en) begin
            if (Q == MOD - 1)
                Q <= {WIDTH{1'b0}};  // Wrap
            else
                Q <= Q + 1'b1;
        end
    end

    assign tc = (Q == MOD - 1) && en;

endmodule
VHDL sync_counter.vhd 
-- Simple N-bit Synchronous Up Counter
library IEEE;
use IEEE.STD_LOGIC_1164.all;
use IEEE.NUMERIC_STD.all;

entity sync_counter is
    generic (WIDTH : integer := 8);
    port (
        clk   : in  std_logic;
        rst_n : in  std_logic;
        en    : in  std_logic;
        Q     : out std_logic_vector(WIDTH-1 downto 0)
    );
end sync_counter;

architecture behavioral of sync_counter is
    signal count : unsigned(WIDTH-1 downto 0);
begin
    process(clk, rst_n)
    begin
        if rst_n = '0' then
            count <= (others => '0');
        elsif rising_edge(clk) then
            if en = '1' then
                count <= count + 1;
            end if;
        end if;
    end process;

    Q <= std_logic_vector(count);
end behavioral;

-- Up/Down Counter with Load
library IEEE;
use IEEE.STD_LOGIC_1164.all;
use IEEE.NUMERIC_STD.all;

entity updown_counter is
    generic (WIDTH : integer := 8);
    port (
        clk   : in  std_logic;
        rst_n : in  std_logic;
        en    : in  std_logic;
        dir   : in  std_logic;  -- 0=up, 1=down
        load  : in  std_logic;
        D     : in  std_logic_vector(WIDTH-1 downto 0);
        Q     : out std_logic_vector(WIDTH-1 downto 0);
        tc    : out std_logic
    );
end updown_counter;

architecture behavioral of updown_counter is
    signal count : unsigned(WIDTH-1 downto 0);
begin
    process(clk, rst_n)
    begin
        if rst_n = '0' then
            count <= (others => '0');
        elsif rising_edge(clk) then
            if load = '1' then
                count <= unsigned(D);
            elsif en = '1' then
                if dir = '1' then
                    count <= count - 1;
                else
                    count <= count + 1;
                end if;
            end if;
        end if;
    end process;

    Q  <= std_logic_vector(count);
    tc <= '1' when (dir = '1' and count = 0) or
                   (dir = '0' and count = (2**WIDTH - 1))
           else '0';
end behavioral;

Async vs Sync Counter Summary

#
Property Asynchronous Synchronous
Clocking Ripple (Q clocks next FF) Common clock to all FFs
Output timing Staggered (glitchy) Simultaneous (clean)
Max frequency Lower (N × tpd) Higher (1 × tpd + logic)
Hardware Simple (no carry logic) More complex
Power Lower (fewer transitions) Higher
Use case Simple dividers, low-speed All modern synchronous designs
