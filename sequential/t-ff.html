T Flip-Flop

The T (Toggle) flip-flop has a single input: when T=1, the output toggles on each clock edge. When T=0, the output holds. It's the simplest flip-flop for building binary counters.

Inputs

T (Toggle), CLK

Outputs

Q, Q'

Trigger

Edge-triggered

Primary use

Binary counters

Derived from

JK FF (J=K=T)

T stands for

Toggle

Truth Table

#
T Qnext Action
0QHold (No change)
1Q'Toggle

Characteristic Equation

Qnext = T ⊕ Q = T'·Q + T·Q'

XOR relationship: The T flip-flop is essentially an XOR of T and the current state. This is why XOR gates appear everywhere in counter logic.

Implementation from Other Flip-Flops

#

T flip-flops are typically not standalone cells. They're built from D or JK flip-flops:

From JK Flip-Flop

J = K = T

Connect T to both J and K inputs

From D Flip-Flop

D = T ⊕ Q

XOR T with Q feedback, or use D = T ? Q' : Q

Building Counters

#

The T flip-flop is ideal for binary counters. With T=1 permanently, each flip-flop divides the frequency by 2:

Ripple Counter (Async)

Q of each stage clocks the next. Simple but has cumulative delay — not for high-speed.

Synchronous Counter

All FFs share same clock. T input logic determines when each bit toggles. Faster but more complex.

RTL (Verilog / VHDL)

#
Verilog t_ff.v 
// T Flip-Flop
module t_ff (
    input  wire clk,
    input  wire rst_n,
    input  wire T,
    output reg  Q
);

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            Q <= 1'b0;
        else if (T)
            Q <= ~Q;     // Toggle
        // else Hold (implicit)
    end

endmodule

// 4-bit Synchronous Counter using T concept
module counter_4bit (
    input  wire clk,
    input  wire rst_n,
    input  wire en,
    output reg [3:0] Q
);

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            Q <= 4'b0;
        else if (en)
            Q <= Q + 1'b1;  // Counts on enable
    end

endmodule
VHDL t_ff.vhd 
-- T Flip-Flop
library IEEE;
use IEEE.STD_LOGIC_1164.all;

entity t_ff is
    port (
        clk   : in  std_logic;
        rst_n : in  std_logic;
        T     : in  std_logic;
        Q     : out std_logic
    );
end t_ff;

architecture behavioral of t_ff is
    signal Q_int : std_logic := '0';
begin
    process(clk, rst_n)
    begin
        if rst_n = '0' then
            Q_int <= '0';
        elsif rising_edge(clk) then
            if T = '1' then
                Q_int <= not Q_int;  -- Toggle
            end if;
        end if;
    end process;

    Q <= Q_int;
end behavioral;
