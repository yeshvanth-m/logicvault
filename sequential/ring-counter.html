Sequential Counter

Ring Counter & Johnson Counter

Shift register-based counters where the output feeds back to the input. Ring counters circulate a single bit; Johnson counters circulate the complement, doubling the state count.

Ring Counter

N states from N FFs

Johnson Counter

2N states from N FFs

Decoding

Simple (1 or 2 gate)

Self-starting

No (needs init)

Ring Counter

#

A shift register where the last output (Qn-1) connects directly back to the first input. A single '1' bit circulates through N positions, creating N unique states.

Clock Q3Q2Q1Q0
Init0001
10010
20100
31000
40001

Not self-starting: If all FFs start at 0, the counter stays at 0 forever. Must initialize with exactly one '1' bit (or use self-correcting logic).

Johnson Counter (Twisted Ring)

#

The complement of the last output (Q'n-1) feeds back to the input. This doubles the number of states to 2N from N flip-flops.

Clock Q3Q2Q1Q0 State
Init00000
100011
200112
301113
411114
511105
611006
710007
800000

State Decoding (only 2 adjacent bits needed)

State 0: Q0' · Q3'   |   State 3: Q2 · Q3'   |   State 4: Q3 · Q0

Ring vs Johnson vs Binary

#
Property Ring Johnson Binary
States from N FFs N 2N 2N
FFs for 8 states 8 4 3

Glitch-free Yes Yes No (needs Gray)
Self-correcting No No (can add) Yes

RTL (Verilog / VHDL)

#
Verilog ring_counter.v 
// Ring Counter - N states
module ring_counter #(
    parameter WIDTH = 4
)(
    input  wire clk,
    input  wire rst_n,
    output reg [WIDTH-1:0] Q
);

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            Q <= {{(WIDTH-1){1'b0}}, 1'b1};  // Init: 0001
        else
            Q <= {Q[WIDTH-2:0], Q[WIDTH-1]};  // Rotate left
    end

endmodule

// Johnson Counter - 2N states
module johnson_counter #(
    parameter WIDTH = 4
)(
    input  wire clk,
    input  wire rst_n,
    output reg [WIDTH-1:0] Q
);

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            Q <= {WIDTH{1'b0}};  // Init: 0000
        else
            Q <= {Q[WIDTH-2:0], ~Q[WIDTH-1]};  // Shift + invert feedback
    end

endmodule

// Johnson Counter with state decode outputs
module johnson_decode #(
    parameter WIDTH = 4  // Produces 2*WIDTH decoded outputs
)(
    input  wire clk,
    input  wire rst_n,
    output reg [WIDTH-1:0] Q,
    output wire [2*WIDTH-1:0] state  // One-hot decoded
);

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            Q <= {WIDTH{1'b0}};
        else
            Q <= {Q[WIDTH-2:0], ~Q[WIDTH-1]};
    end

    // Decode: each state uses 2 adjacent bits
    assign state[0] = ~Q[0] & ~Q[WIDTH-1];  // 0000
    assign state[1] =  Q[0] & ~Q[1];         // 0001
    assign state[2] =  Q[1] & ~Q[2];         // 0011
    assign state[3] =  Q[2] & ~Q[3];         // 0111
    assign state[4] =  Q[3] &  Q[0];         // 1111
    assign state[5] = ~Q[0] &  Q[1];         // 1110
    assign state[6] = ~Q[1] &  Q[2];         // 1100
    assign state[7] = ~Q[2] &  Q[3];         // 1000

endmodule
VHDL ring_counter.vhd 
-- Ring Counter - N states
library IEEE;
use IEEE.STD_LOGIC_1164.all;

entity ring_counter is
    generic (WIDTH : integer := 4);
    port (
        clk   : in  std_logic;
        rst_n : in  std_logic;
        Q     : out std_logic_vector(WIDTH-1 downto 0)
    );
end ring_counter;

architecture behavioral of ring_counter is
    signal Q_int : std_logic_vector(WIDTH-1 downto 0);
begin
    process(clk, rst_n)
    begin
        if rst_n = '0' then
            Q_int <= (0 => '1', others => '0');  -- Init: 0001
        elsif rising_edge(clk) then
            Q_int <= Q_int(WIDTH-2 downto 0) & Q_int(WIDTH-1);
        end if;
    end process;

    Q <= Q_int;
end behavioral;

-- Johnson Counter - 2N states
library IEEE;
use IEEE.STD_LOGIC_1164.all;

entity johnson_counter is
    generic (WIDTH : integer := 4);
    port (
        clk   : in  std_logic;
        rst_n : in  std_logic;
        Q     : out std_logic_vector(WIDTH-1 downto 0)
    );
end johnson_counter;

architecture behavioral of johnson_counter is
    signal Q_int : std_logic_vector(WIDTH-1 downto 0);
begin
    process(clk, rst_n)
    begin
        if rst_n = '0' then
            Q_int <= (others => '0');
        elsif rising_edge(clk) then
            Q_int <= Q_int(WIDTH-2 downto 0) & (not Q_int(WIDTH-1));
        end if;
    end process;

    Q <= Q_int;
end behavioral;

Applications

#

LED Sequencing

Ring counters create "chasing" LED patterns. Each output directly drives an LED.

Phase Generation

Johnson counters generate multi-phase clocks with 50% duty cycle.

Stepper Motor

Ring counter outputs directly drive stepper motor coil sequences.

State Machines

One-hot encoding in FSMs — fast decode, easy debug.
