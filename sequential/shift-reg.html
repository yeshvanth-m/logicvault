Shift Register

A shift register is a cascade of flip-flops where the output of one feeds the input of the next. Data "shifts" one position per clock cycle. Used for serial-parallel conversion, delay lines, and data buffering.

Building block

D Flip-Flops

N-bit register

N flip-flops

Shift per clock

1 bit position

Direction

Left / Right / Both

SISO (Serial-In Serial-Out)

#

The simplest shift register. Data enters one bit at a time and exits one bit at a time after N clock cycles (where N is the register width). Acts as a delay line.

Delay = N clock cycles

Dout[t] = Din[t - N]

SIPO (Serial-In Parallel-Out)

#

Data enters serially but all flip-flop outputs are available simultaneously. After N clocks, the complete N-bit word is available in parallel. Used in serial receivers (UART, SPI).

PISO (Parallel-In Serial-Out)

#

An N-bit word is loaded in parallel (all at once), then shifted out serially one bit per clock. Used in serial transmitters (UART TX, SPI MOSI).

Mode control: PISO typically has a LOAD/SHIFT control signal. When LOAD=1, parallel data is captured. When LOAD=0, data shifts out serially.

PIPO (Parallel-In Parallel-Out)

#

All bits loaded and output in parallel. Essentially a simple register or pipeline stage. Data is captured on clock edge and held until next load.

Function

Q[n:0] ← D[n:0]

On each clock edge

Use Cases

Pipeline registers

Temporary storage, data buffering

Bidirectional Shift Register

#

Can shift data left or right based on a direction control signal. Uses multiplexers to select between left neighbor and right neighbor for each flip-flop's input.

DIR SHIFT Operation
00Hold (no change)
01Shift Right
10Hold (no change)
11Shift Left

Arithmetic shifts: Shift left = multiply by 2. Shift right (arithmetic) = divide by 2. Logical shift fills with 0; arithmetic shift preserves sign bit.

RTL (Verilog / VHDL)

#
Verilog shift_reg.v 
// 8-bit Shift Register (SIPO)
module shift_reg_sipo #(
    parameter WIDTH = 8
)(
    input  wire clk,
    input  wire rst_n,
    input  wire din,           // Serial input
    output wire [WIDTH-1:0] Q  // Parallel output
);

    reg [WIDTH-1:0] shift;

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            shift <= {WIDTH{1'b0}};
        else
            shift <= {shift[WIDTH-2:0], din};  // Shift left, new bit enters LSB
    end

    assign Q = shift;

endmodule

// 8-bit PISO Shift Register
module shift_reg_piso #(
    parameter WIDTH = 8
)(
    input  wire clk,
    input  wire rst_n,
    input  wire load,          // 1=load parallel, 0=shift
    input  wire [WIDTH-1:0] D, // Parallel input
    output wire dout           // Serial output (MSB first)
);

    reg [WIDTH-1:0] shift;

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            shift <= {WIDTH{1'b0}};
        else if (load)
            shift <= D;                       // Load parallel
        else
            shift <= {shift[WIDTH-2:0], 1'b0}; // Shift left
    end

    assign dout = shift[WIDTH-1];  // MSB is serial out

endmodule
VHDL shift_reg.vhd 
-- 8-bit Shift Register (SIPO)
library IEEE;
use IEEE.STD_LOGIC_1164.all;

entity shift_reg_sipo is
    generic (WIDTH : integer := 8);
    port (
        clk   : in  std_logic;
        rst_n : in  std_logic;
        din   : in  std_logic;
        Q     : out std_logic_vector(WIDTH-1 downto 0)
    );
end shift_reg_sipo;

architecture behavioral of shift_reg_sipo is
    signal shift : std_logic_vector(WIDTH-1 downto 0);
begin
    process(clk, rst_n)
    begin
        if rst_n = '0' then
            shift <= (others => '0');
        elsif rising_edge(clk) then
            shift <= shift(WIDTH-2 downto 0) & din;
        end if;
    end process;

    Q <= shift;
end behavioral;

Applications

#

Serial Communication

UART, SPI, I2C all use shift registers for serial-parallel conversion.

Delay Lines

SISO creates precise N-cycle delays. Used in DSP, audio effects.

LED Drivers

Cascade shift registers (74HC595) to drive many LEDs with few pins.

CRC/Checksum

LFSR-based shift registers compute CRC for error detection.
