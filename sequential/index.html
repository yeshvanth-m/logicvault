Sequential Memory Elements

Sequential Logic

Sequential circuits have memory — their outputs depend on both current inputs and past history. Unlike combinational circuits, they require clock signals and can store state.

Combinational vs Sequential

Output = f(inputs, state)

Synchronous

Latches Level-sensitive

SR

SR Latch

Set-Reset

The simplest memory element. Set and Reset inputs control state. Has a forbidden state (S=R=1).
D

D Latch

Data / Transparent

Single data input eliminates forbidden state. Transparent when enabled — output follows input.

Flip-Flops Edge-triggered

D

D Flip-Flop

Data FF

The workhorse of synchronous design. Samples D only at clock edge. Foundation of registers and pipelines.
JK

JK Flip-Flop

Universal FF

Most versatile flip-flop with toggle mode (J=K=1). No forbidden state. Used in counters and academic contexts.
T

T Flip-Flop

Toggle FF

Toggles output when T=1, holds when T=0. Simplest structure for binary counters and frequency dividers.

Latch vs Flip-Flop

#
Property Latch Flip-Flop
Trigger type Level-sensitive Edge-triggered
When sampled Entire enable HIGH period Only at clock edge
Transparency Yes (can cause races) No
Gate count Lower (~4 gates) Higher (~8-12 gates)
Modern usage Special cases, CDC Standard synchronous design

Industry standard: Flip-flops (especially D flip-flops) dominate synchronous design. Latches appear in special cases like clock gating cells and CDC synchronizers.
