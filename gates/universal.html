Fundamental Concept NAND & NOR

Universal Gates

A universal gate can implement any Boolean function using only copies of itself. NAND and NOR are the two universal gates — entire processors can be built using just one of them.

What Makes a Gate Universal?

#

A gate is universal (or functionally complete) if it can implement the three fundamental Boolean operations:

NOT (Inversion)

A'

AND (Conjunction)

A · B

OR (Disjunction)

A + B

With these three operations, you can express any Boolean function (this is called functional completeness). AND, OR, NOT individually are NOT universal — but NAND alone, or NOR alone, can create all three.

The power of universal gates: Manufacturing just one gate type simplifies IC fabrication. The Apollo Guidance Computer used ~5,600 NOR gates exclusively.

Implementations Using NAND

#

Every basic gate can be constructed using only NAND gates:

NOT from NAND 1 gate

A' = (A · A)'

Connect both inputs of a NAND gate together. When A=0, output=1. When A=1, output=0.

AND from NAND 2 gates

A · B = ((A · B)')'

NAND the inputs, then invert the result using another NAND (with tied inputs).

OR from NAND 3 gates

A + B = (A' · B')' = ((A·A)' · (B·B)')'

By De Morgan's law: invert both inputs (2 NANDs as inverters), then NAND the results.

BUF from NAND 2 gates

A = ((A·A)'·(A·A)')' = (A')'

Two inverters in series: invert then invert again. Output equals input.

Implementations Using NOR

#

Every basic gate can also be constructed using only NOR gates:

NOT from NOR 1 gate

A' = (A + A)'

Connect both inputs of a NOR gate together. Same principle as NAND inverter.

OR from NOR 2 gates

A + B = ((A + B)')'

NOR the inputs, then invert the result using another NOR (with tied inputs).

AND from NOR 3 gates

A · B = (A' + B')' = ((A+A)' + (B+B)')'

By De Morgan's law: invert both inputs (2 NORs as inverters), then NOR the results.

BUF from NOR 2 gates

A = ((A+A)'+(A+A)')' = (A')'

Two inverters in series: invert then invert again.

XOR from Universal Gates

#

XOR is critical for arithmetic (adders) and parity. Here's how to build it:

XOR from NAND 4 gates

A ⊕ B = ((A·(A·B)')·(B·(A·B)')'

Classic 4-NAND XOR: NAND A&B, then NAND result with A and with B separately, then NAND those outputs.

XOR from NOR 5 gates

A ⊕ B = ((A+B)' + (A'+B')')'

Requires more gates than NAND implementation due to NOR's structure.

Why NAND Wins in CMOS

#

Despite both being universal, NAND is strongly preferred in modern CMOS design:

NAND Advantage

Parallel PMOS

PMOS transistors in parallel = fast pull-up to VDD. Series NMOS handles pull-down.

NOR Disadvantage

Series PMOS

PMOS transistors in series = slow pull-up. PMOS is already ~2-3× slower than NMOS.

Result

NAND 2× faster

Typical 2-input NAND is roughly twice as fast as equivalent NOR in same process.

Standard cell libraries: Modern cell libraries are NAND-heavy. Complex cells like AOI21 (AND-OR-Invert) and OAI21 (OR-AND-Invert) are optimized around NAND-style topology.

Practical Applications

#

Apollo Guidance Computer (1966)

Built entirely from ~5,600 3-input NOR gates. NOR was chosen because it was more reliable to manufacture with early IC technology. Redundancy and simplicity trumped speed.

Modern ASIC/FPGA

Technology mapping converts RTL into NAND-dominant standard cells. Synthesis tools automatically restructure logic to minimize area and maximize speed using NAND-based cells.

Flash Memory (NAND Flash)

NAND flash uses series-connected transistors (like NAND gate's NMOS network). This allows higher density than NOR flash, which uses parallel transistors.

TTL Logic Families

7400 series: 7400 = Quad NAND, 7402 = Quad NOR. NAND gates are more common in TTL because they're faster and cheaper to manufacture.
