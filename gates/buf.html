Buffer (BUF)

A buffer is a non-inverting driver used to increase drive strength or isolate signal paths. Implemented as two cascaded inverters.

Inputs

1 (A)

Outputs

1 (Y)

Transistor count

4 (2x INV)

Logic depth

2 levels

Boolean Expression

#

Key insight: Buffers are essential for driving large capacitive loads and are often sized exponentially in multi-stage chains.

Circuit Diagram

#

CMOS Implementation: A buffer is implemented as two inverters in series. The second inverter restores the original polarity while providing increased drive strength.

RTL (Verilog / VHDL)

#
Verilog buf_gate.v 
module buf_gate (
    input  a,
    output y
);
    assign y = a;
endmodule
VHDL buf_gate.vhd 
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;

entity buf_gate is
    port (
        a : in  STD_LOGIC;
        y : out STD_LOGIC
    );
end entity;

architecture dataflow of buf_gate is
begin
    y <= a;
end architecture;
