NAND Gate

The NAND gate outputs 0 only when all inputs are 1. It is a universal gate and is preferred in CMOS design due to optimal transistor arrangement.

Key insight: NAND gates dominate CMOS standard cell libraries because PMOS in parallel is much faster than PMOS in series.

Why NAND is Universal

#

A universal gate can implement any Boolean function using only that gate type. NAND achieves this because it can construct NOT, AND, and OR — the complete Boolean basis:

NOT from NAND

A' = (A · A)'

Connect both inputs together

AND from NAND

A · B = ((A · B)')'

NAND followed by NOT (2 gates)

OR from NAND

A + B = (A' · B')'

De Morgan: invert inputs, then NAND (3 gates)

Practical implication: Entire processors can be built using only NAND gates. Early computers and the Apollo Guidance Computer used exclusively NOR gates for similar reasons.

Circuit Diagram

#

CMOS Implementation: NAND uses PMOS in parallel (fast pull-up) and NMOS in series (pull-down network). This arrangement is faster than NOR's series PMOS.

RTL (Verilog / VHDL)

#
Verilog nand_gate.v 
module nand_gate (
    input  a,
    input  b,
    output y
);
    assign y = ~(a & b);
endmodule
VHDL nand_gate.vhd 
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;

entity nand_gate is
    port (
        a : in  STD_LOGIC;
        b : in  STD_LOGIC;
        y : out STD_LOGIC
    );
end entity;

architecture dataflow of nand_gate is
begin
    y <= a nand b;
end architecture;
