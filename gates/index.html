Fundamentals 8 Gates

Logic Gates

Logic gates are the fundamental building blocks of digital circuits. Each gate performs a basic Boolean operation on one or more binary inputs to produce a single binary output.

Basic Gates 3 gates

A·B

AND Gate

Conjunction

Output is 1 only when ALL inputs are 1. Implements logical conjunction.

00→0, 01→0, 10→0, 11→1
A+B

OR Gate

Disjunction

Output is 1 when ANY input is 1. Implements logical disjunction.

00→0, 01→1, 10→1, 11→1
A'

NOT Gate

Inverter

Inverts the input. Output is opposite of input. Also called inverter.

0→1, 1→0

Buffer 1 gate

A

Buffer (BUF)

Identity

Passes input to output unchanged. Used for signal strengthening and timing.

0→0, 1→1

Universal Gates 2 gates

(A·B)'

NAND Gate

NOT-AND

Output is 0 only when ALL inputs are 1. Universal gate — can build any logic.

00→1, 01→1, 10→1, 11→0
(A+B)'

NOR Gate

NOT-OR

Output is 1 only when ALL inputs are 0. Universal gate — can build any logic.

00→1, 01→0, 10→0, 11→0

Exclusive Gates 2 gates

A⊕B

XOR Gate

Exclusive OR

Output is 1 when inputs DIFFER. Essential for arithmetic and parity circuits.

00→0, 01→1, 10→1, 11→0
A⊙B

XNOR Gate

Equivalence

Output is 1 when inputs are SAME. Also called equivalence or coincidence gate.

00→1, 01→0, 10→0, 11→1
