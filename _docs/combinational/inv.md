# Inverter (INV)

Description: Single-input logic that outputs the logical complement of input.

Truth table:

| A | Y |
|---:|---:|
| 0 | 1 |
| 1 | 0 |

Notes:
- Standard CMOS: PMOS pull-up, NMOS pull-down.
- Key metrics: switching threshold, VTC, rise/fall strengths.
- Used as a building block and buffer (sized variants).
