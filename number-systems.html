Number Systems

Digital systems use different number bases. Understanding binary, octal, decimal, and hexadecimal representations is essential for working with digital logic and computer systems.

Example: 10110101₂ to Decimal

1×128 + 0×64 + 1×32 + 1×16 + 0×8 + 1×4 + 0×2 + 1×1
= 128 + 32 + 16 + 4 + 1
= 181₁₀

Bit positions: Bits are numbered from right (LSB = bit 0) to left (MSB). An 8-bit value uses bits 0–7.

Conversion Methods

#

Decimal → Binary (Repeated Division)

181 ÷ 2 = 90 remainder 1 (LSB)
90 ÷ 2 = 45 remainder 0
45 ÷ 2 = 22 remainder 1
22 ÷ 2 = 11 remainder 0
11 ÷ 2 = 5 remainder 1
5 ÷ 2 = 2 remainder 1
2 ÷ 2 = 1 remainder 0
1 ÷ 2 = 0 remainder 1 (MSB)
Read up: 181₁₀ = 10110101₂

Binary → Hex (Group by 4)

10110101₂
= 1011 | 0101 (group from right)
= B | 5
= 0xB5

Binary → Octal (Group by 3)

10110101₂
= 10 | 110 | 101 (group from right, pad left)
= 010 | 110 | 101
= 2 | 6 | 5
= 265₈
