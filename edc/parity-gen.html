Overview

A Parity Generator is a combinational circuit that computes a single parity bit from a group of data bits. The parity bit is appended to the data for transmission or storage, enabling the receiver to detect single-bit errors.

The circuit is fundamentally built using XOR gates. For even parity, the parity bit is set so that the total count of 1s in (data + parity) is even. For odd parity, the count is made odd. This simple scheme cannot correct errors or detect multiple errors.

Parity Types

Even Parity

Parity bit makes total number of 1s even. Most common in computer systems.

P = D₀ ⊕ D₁ ⊕ D₂ ⊕ ... ⊕ Dₙ₋₁
Odd Parity

Parity bit makes total number of 1s odd. Used in some serial protocols.

P = D₀ ⊕ D₁ ⊕ D₂ ⊕ ... ⊕ Dₙ₋₁ ⊕ 1
Horizontal Parity

One parity bit per data word. Transmitted alongside each byte.

Row-wise parity
Vertical Parity

Parity computed across corresponding bits of multiple words.

Column-wise parity

Example: Even Parity Generation

Data: 1 0 1 1 0 0 1 → Count of 1s = 4 (even)
With Parity: 1 0 1 1 0 0 1 0 → P=0 keeps count even

Example: Odd Parity Generation

Data: 1 0 1 1 0 0 1 → Count of 1s = 4 (even)
With Parity: 1 0 1 1 0 0 1 1 → P=1 makes count odd

Boolean Equations

P_even = D₀ ⊕ D₁ ⊕ D₂ ⊕ ... ⊕ Dₙ₋₁
Odd Parity Generator (n-bit)
P_odd = D₀ ⊕ D₁ ⊕ D₂ ⊕ ... ⊕ Dₙ₋₁ ⊕ 1
4-bit Even Parity
P = D₃ ⊕ D₂ ⊕ D₁ ⊕ D₀

Why XOR Works for Parity

XOR outputs 1 when odd number of inputs are 1 XOR outputs 0 when even number of inputs are 1 So XOR of all bits gives: - 1 if odd count of 1s → need P=1 to make even total - 0 if even count of 1s → need P=0 to keep even total Result: P_even = XOR(all data bits) ✓

RTL Implementation

VERILOG parity_gen.v 
// Parameterized Parity Generator
module parity_gen #(
    parameter WIDTH = 8,
    parameter ODD   = 0   // 0=even, 1=odd
)(
    input  [WIDTH-1:0] data,
    output             parity
);

// XOR reduction operator
assign parity = ^data ^ ODD;

endmodule

// 8-bit Even Parity Generator (explicit)
module parity_gen_8bit (
    input  [7:0] data,
    output       parity
);

// Tree structure for minimum delay
wire p01, p23, p45, p67;
wire p0123, p4567;

// Level 1
assign p01 = data[0] ^ data[1];
assign p23 = data[2] ^ data[3];
assign p45 = data[4] ^ data[5];
assign p67 = data[6] ^ data[7];

// Level 2
assign p0123 = p01 ^ p23;
assign p4567 = p45 ^ p67;

// Level 3
assign parity = p0123 ^ p4567;

endmodule

// Combined generator with data output
module parity_gen_with_data #(
    parameter WIDTH = 8
)(
    input  [WIDTH-1:0] data_in,
    output [WIDTH:0]   data_out  // data + parity
);

wire parity;

parity_gen #(.WIDTH(WIDTH)) gen (
    .data(data_in),
    .parity(parity)
);

assign data_out = {parity, data_in};

endmodule
VHDL parity_gen.vhd 
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;

entity parity_gen is
    generic (
        WIDTH : natural := 8;
        ODD   : std_logic := '0'
    );
    port (
        data   : in  std_logic_vector(WIDTH-1 downto 0);
        parity : out std_logic
    );
end parity_gen;

architecture behavioral of parity_gen is
begin
    process(data)
        variable p : std_logic;
    begin
        p := ODD;
        for i in 0 to WIDTH-1 loop
            p := p xor data(i);
        end loop;
        parity <= p;
    end process;
end behavioral;

Applications

Memory Systems

Parity bits stored with each memory word. DRAM often uses 9 bits (8 data + 1 parity).

Simple error detection
Serial Communication

UART frames include optional parity bit. RS-232, RS-485 protocols.

Even/Odd/None parity
Bus Transfers

Parity lanes on PCI, PCI Express for data integrity.

Per-byte parity
RAID Storage

XOR-based parity used in RAID 5/6 for disk failure recovery.

Block-level XOR

Interview Notes

  • Parity generator is XOR of all data bits. Even parity: P = ⊕D[i]. Odd parity: P = ⊕D[i] ⊕ 1.
  • Detects all single-bit errors. Cannot detect 2-bit errors (they cancel out). Cannot correct errors.
  • For n-bit data: need (n-1) XOR gates in linear chain, or log₂(n) levels in tree structure.
  • Tree structure reduces delay from O(n) to O(log n). Important for high-speed systems.
  • Verilog reduction operator: ^data computes XOR of all bits. Clean, synthesizable.
  • 74180/74280 are standard parity generator/checker ICs with 8-bit inputs.
