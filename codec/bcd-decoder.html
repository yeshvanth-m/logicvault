Overview

A BCD Decoder (Binary-Coded Decimal to Decimal Decoder) converts a 4-bit BCD input into 10 output lines, one for each decimal digit (0-9). It's a specialized form of a 4-to-16 decoder that only produces valid outputs for the BCD range (0000 to 1001).

BCD is widely used in applications where decimal numbers need to be displayed or processed digit-by-digit, such as digital clocks, calculators, and instrumentation. Each decimal digit is represented by its 4-bit binary equivalent.

BCD Encoding

What is BCD?

Each decimal digit (0-9) encoded as 4-bit binary. NOT the same as binary.

Decimal 25 = 0010 0101 BCD
Why BCD?

Easy decimal display, human-readable, no binary-to-decimal conversion.

Used in meters, clocks
BCD vs Binary

BCD uses more bits but simplifies decimal I/O.

99 = 1001 1001 (BCD) vs 1100011 (bin)
Invalid Codes

1010-1111 are not valid BCD. 6 unused states.

Don't care or error detect

RTL Implementation

VERILOG bcd_decoder.v 
// BCD to Decimal Decoder
module bcd_decoder (
    input      [3:0] bcd,
    output reg [9:0] decimal,
    output           invalid
);

// Invalid BCD detection (>9)
assign invalid = (bcd[3] & bcd[2]) | (bcd[3] & bcd[1]);

always @(*) begin
    decimal = 10'b0;
    case (bcd)
        4'd0: decimal = 10'b0000000001;
        4'd1: decimal = 10'b0000000010;
        4'd2: decimal = 10'b0000000100;
        4'd3: decimal = 10'b0000001000;
        4'd4: decimal = 10'b0000010000;
        4'd5: decimal = 10'b0000100000;
        4'd6: decimal = 10'b0001000000;
        4'd7: decimal = 10'b0010000000;
        4'd8: decimal = 10'b0100000000;
        4'd9: decimal = 10'b1000000000;
        default: decimal = 10'b0; // Invalid BCD
    endcase
end

endmodule

// Alternative: Shift-based one-hot decoder
module bcd_decoder_shift (
    input  [3:0] bcd,
    output [9:0] decimal,
    output       invalid
);

assign invalid = (bcd > 4'd9);
assign decimal = invalid ? 10'b0 : (10'b1 << bcd);

endmodule

// Multi-digit BCD decoder
module bcd_2digit_decoder (
    input  [7:0] bcd,     // 2-digit BCD (00-99)
    output [9:0] tens,    // Tens digit decoded
    output [9:0] units,   // Units digit decoded
    output       invalid
);

wire inv_tens, inv_units;

bcd_decoder dec_tens (
    .bcd(bcd[7:4]),
    .decimal(tens),
    .invalid(inv_tens)
);

bcd_decoder dec_units (
    .bcd(bcd[3:0]),
    .decimal(units),
    .invalid(inv_units)
);

assign invalid = inv_tens | inv_units;

endmodule
VHDL bcd_decoder.vhd 
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;
use IEEE.NUMERIC_STD.ALL;

entity bcd_decoder is
    port (
        bcd     : in  std_logic_vector(3 downto 0);
        decimal : out std_logic_vector(9 downto 0);
        invalid : out std_logic
    );
end bcd_decoder;

architecture behavioral of bcd_decoder is
begin
    invalid <= '1' when unsigned(bcd) > 9 else '0';
    
    process(bcd)
    begin
        decimal <= (others => '0');
        if unsigned(bcd) <= 9 then
            decimal(to_integer(unsigned(bcd))) <= '1';
        end if;
    end process;
end behavioral;

Interview Notes

  • BCD: Each decimal digit (0-9) encoded as 4 bits. Decimal 42 = 0100 0010 in BCD.
  • BCD decoder: 4-to-10 decoder. Subset of 4-to-16, only 10 outputs for valid BCD.
  • Invalid codes: 1010-1111 are not valid BCD. Detect with D₃·D₂ + D₃·D₁.
  • BCD advantages: Easy decimal display, no binary-decimal conversion needed.
  • BCD disadvantages: Wastes 6 states per digit, less efficient than binary.
  • Common cascade: BCD counter (7490) → BCD decoder (7442) → display driver.
