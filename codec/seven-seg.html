Overview

A 7-Segment Decoder converts a 4-bit binary or BCD input into seven output signals that control the individual segments of a 7-segment LED or LCD display. Each segment (labeled a through g) is lit or unlit based on the digit being displayed.

This decoder is ubiquitous in digital systems: calculators, digital clocks, multimeters, scoreboards, and any device that needs to display numeric information. The 7447 and 7448 are classic TTL decoders, while CD4511 is the CMOS equivalent.

Segment positions:
a = top, b = upper-right, c = lower-right
d = bottom, e = lower-left, f = upper-left
g = middle, dp = decimal point
Common Cathode

All cathodes connected. Segments lit by HIGH signal. Most intuitive.

7448, CD4511
Common Anode

All anodes connected. Segments lit by LOW signal. Active-low outputs.

7447
Typical Current

10-20 mA per segment. Use current-limiting resistors.

R ≈ (Vcc - Vf) / If
Multiplexing

Multiple digits share segments. Rapidly scan through digits.

Reduces pin count

RTL Implementation

VERILOG seven_seg.v 
// 7-Segment Decoder (Common Cathode, Active-High)
module seven_seg_decoder (
    input      [3:0] digit,
    output reg [6:0] seg  // seg[6:0] = {g,f,e,d,c,b,a}
);

// Segment encoding: gfedcba
always @(*) begin
    case (digit)
        4'h0: seg = 7'b0111111;  // 0
        4'h1: seg = 7'b0000110;  // 1
        4'h2: seg = 7'b1011011;  // 2
        4'h3: seg = 7'b1001111;  // 3
        4'h4: seg = 7'b1100110;  // 4
        4'h5: seg = 7'b1101101;  // 5
        4'h6: seg = 7'b1111101;  // 6
        4'h7: seg = 7'b0000111;  // 7
        4'h8: seg = 7'b1111111;  // 8
        4'h9: seg = 7'b1101111;  // 9
        4'hA: seg = 7'b1110111;  // A
        4'hB: seg = 7'b1111100;  // b
        4'hC: seg = 7'b0111001;  // C
        4'hD: seg = 7'b1011110;  // d
        4'hE: seg = 7'b1111001;  // E
        4'hF: seg = 7'b1110001;  // F
    endcase
end

endmodule

// Common Anode (Active-Low) version
module seven_seg_decoder_ca (
    input      [3:0] digit,
    output     [6:0] seg_n  // Active-low outputs
);

wire [6:0] seg_cc;

seven_seg_decoder cc_dec (
    .digit(digit),
    .seg(seg_cc)
);

assign seg_n = ~seg_cc;  // Invert for common anode

endmodule

// Multiplexed 4-digit display
module four_digit_display (
    input        clk,
    input        rst_n,
    input [15:0] value,     // 4 BCD digits
    output [6:0] seg,        // Shared segments
    output [3:0] digit_sel   // Digit select (active-low)
);

reg [1:0] scan_cnt;
reg [3:0] current_digit;
reg [15:0] refresh_cnt;  // ~1kHz refresh rate

// Refresh counter for digit scanning
always @(posedge clk or negedge rst_n) begin
    if (!rst_n) begin
        refresh_cnt <= 0;
        scan_cnt <= 0;
    end else begin
        refresh_cnt <= refresh_cnt + 1;
        if (refresh_cnt == 0)
            scan_cnt <= scan_cnt + 1;
    end
end

// Select current digit
always @(*) begin
    case (scan_cnt)
        2'd0: current_digit = value[3:0];
        2'd1: current_digit = value[7:4];
        2'd2: current_digit = value[11:8];
        2'd3: current_digit = value[15:12];
    endcase
end

// Digit select (active-low)
assign digit_sel = ~(4'b0001 << scan_cnt);

// Segment decoder
seven_seg_decoder dec (
    .digit(current_digit),
    .seg(seg)
);

endmodule

library IEEE;

    );

Interview Notes

  • 7 segments labeled a-g. 'a' is top, 'g' is middle. Can display 0-9 and hex A-F.
  • Common cathode: segments lit by HIGH. Common anode: segments lit by LOW.
  • Each segment has its own Boolean function. Derive using K-map with don't cares for invalid BCD.
  • Multiplexing: share segment lines, scan digit-select rapidly (~1kHz). Reduces pins.
  • Current limiting: Always use resistors (typically 220Ω-1kΩ) to limit LED current.
  • 7447: Common anode TTL decoder. 7448/CD4511: Common cathode decoders.
