Overview

A Barrel Shifter is a combinational circuit that can shift or rotate an n-bit input by any number of positions (0 to n-1) in a single clock cycle. Unlike sequential shift registers that shift one bit per cycle, a barrel shifter uses a logarithmic arrangement of multiplexers to achieve constant-time shifting.

The key insight is that any shift amount can be decomposed into powers of 2. For example, shifting by 5 positions equals shifting by 4 + 1. An 8-bit barrel shifter uses 3 stages that can shift by 1, 2, and 4 positions respectively, controlled by the 3 bits of the shift amount.

Shift Operations

Logical Left Shift (SLL)

Shift bits left, fill right with zeros. Equivalent to multiply by 2ⁿ.

10110100 << 2 = 11010000
Logical Right Shift (SRL)

Shift bits right, fill left with zeros. Unsigned divide by 2ⁿ.

10110100 >> 2 = 00101101
Arithmetic Right Shift (SRA)

Shift right, replicate sign bit. Signed divide by 2ⁿ.

10110100 >>> 2 = 11101101
Rotate Left (ROL)

Bits shifted out left wrap around to right.

10110100 ROL 2 = 11010010
Rotate Right (ROR)

Bits shifted out right wrap around to left.

10110100 ROR 2 = 00101101
Funnel Shift

Shift 2n bits, extract n-bit result. Used for misaligned loads.

{A,B} shifted, extract middle

Worked Example

8-bit Left Shift by 5 (binary: 101)

Input:    A B C D E F G H     (shift amount = 5 = 101₂)

Stage 0 (S[0]=1, shift by 1):
          B C D E F G H 0     ← shifted left by 1

Stage 1 (S[1]=0, shift by 0):
          B C D E F G H 0     ← no change

Stage 2 (S[2]=1, shift by 4):
          F G H 0 0 0 0 0     ← shifted left by 4

Output:   F G H 0 0 0 0 0     = input << 5 ✓

8-bit Rotate Right by 3 (binary: 011)

Input:    A B C D E F G H     (rotate amount = 3 = 011₂)

Stage 0 (S[0]=1, rotate by 1):
          H A B C D E F G     ← bit H wraps to MSB

Stage 1 (S[1]=1, rotate by 2):
          F G H A B C D E     ← 2 more bits wrap

Stage 2 (S[2]=0, rotate by 0):
          F G H A B C D E     ← no change

Output:   F G H A B C D E     = input ROR 3 ✓

RTL Implementation

VERILOG barrel_shifter.v 
// 32-bit Barrel Shifter - All Operations
module barrel_shifter (
    input      [31:0] data_in,
    input      [4:0]  shamt,     // Shift amount (0-31)
    input      [1:0]  op,        // 00=SLL, 01=SRL, 10=SRA, 11=ROR
    output reg [31:0] data_out
);

always @(*) begin
    case (op)
        2'b00: data_out = data_in << shamt;                       // SLL
        2'b01: data_out = data_in >> shamt;                       // SRL
        2'b10: data_out = $signed(data_in) >>> shamt;           // SRA
        2'b11: data_out = (data_in >> shamt) | 
                          (data_in << (32 - shamt));          // ROR
    endcase
end

endmodule

// Explicit Stage-by-Stage Implementation (8-bit)
module barrel_shifter_8bit (
    input      [7:0] data_in,
    input      [2:0] shamt,
    input            left,      // 1=left, 0=right
    input            arith,     // 1=arithmetic (for right)
    output     [7:0] data_out
);

wire [7:0] stage0, stage1, stage2;
wire fill = arith & data_in[7];  // Sign bit for arithmetic

// Reverse input for right shifts
wire [7:0] din = left ? data_in : 
                        {data_in[0],data_in[1],data_in[2],data_in[3],
                         data_in[4],data_in[5],data_in[6],data_in[7]};

// Stage 0: Shift by 1
assign stage0 = shamt[0] ? {din[6:0], 1'b0} : din;

// Stage 1: Shift by 2
assign stage1 = shamt[1] ? {stage0[5:0], 2'b00} : stage0;

// Stage 2: Shift by 4
assign stage2 = shamt[2] ? {stage1[3:0], 4'b0000} : stage1;

// Reverse output for right shifts
wire [7:0] reversed = {stage2[0],stage2[1],stage2[2],stage2[3],
                        stage2[4],stage2[5],stage2[6],stage2[7]};

// Final output with sign extension for arithmetic right shift
wire [7:0] mask = (8'hFF << (8 - shamt));
assign data_out = left ? stage2 : 
                   (arith ? (reversed | (fill ? mask : 8'h00)) : reversed);

endmodule

// Funnel Shifter (for misaligned loads)
module funnel_shifter (
    input      [31:0] high,     // Upper 32 bits
    input      [31:0] low,      // Lower 32 bits
    input      [4:0]  shamt,    // Shift amount
    output     [31:0] result
);

wire [63:0] combined = {high, low};
assign result = combined[31 + shamt -: 32];

endmodule
VHDL barrel_shifter.vhd 
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;
use IEEE.NUMERIC_STD.ALL;

entity barrel_shifter is
    generic (WIDTH : natural := 32);
    port (
        data_in  : in  std_logic_vector(WIDTH-1 downto 0);
        shamt    : in  natural range 0 to WIDTH-1;
        op       : in  std_logic_vector(1 downto 0);
        data_out : out std_logic_vector(WIDTH-1 downto 0)
    );
end barrel_shifter;

architecture behavioral of barrel_shifter is
begin
    process(data_in, shamt, op)
    begin
        case op is
            when "00" =>  -- SLL
                data_out <= std_logic_vector(
                    shift_left(unsigned(data_in), shamt));
            when "01" =>  -- SRL
                data_out <= std_logic_vector(
                    shift_right(unsigned(data_in), shamt));
            when "10" =>  -- SRA
                data_out <= std_logic_vector(
                    shift_right(signed(data_in), shamt));
            when others =>  -- ROR
                data_out <= std_logic_vector(
                    rotate_right(unsigned(data_in), shamt));
        end case;
    end process;
end behavioral;

Variations

Bidirectional Shifter

Reverse input bits, shift left, reverse output for right shift.

Shares shift logic
Cross-bar Shifter

Single stage with n n:1 MUXes. Fastest but O(n²) area.

1 MUX delay, huge area
Logarithmic Shifter

Standard barrel shifter with log₂(n) stages. Best area-delay trade-off.

O(n log n) area
Funnel Shifter

Takes 2n input bits, shifts, outputs middle n bits. Misaligned memory access.

Double-word shift

Interview Notes

  • Barrel shifter uses log₂(n) stages, each stage shifts by power of 2 (1, 2, 4, 8...).
  • Any shift amount decomposed into powers of 2. Shift by 5 = shift by 4 + shift by 1.
  • n-bit shifter needs n × log₂(n) 2:1 MUXes. 32-bit needs 160 MUXes.
  • Right shift: reverse input, left shift, reverse output. Or use separate right-shift logic.
  • Arithmetic vs logical right shift: fill with sign bit (MSB) vs fill with zeros.
  • Funnel shifter for misaligned loads: concatenate two words, shift, extract middle word.
