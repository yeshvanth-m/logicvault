Overview

Popcount (Population Count), also known as Hamming Weight or Sideways Sum, counts the number of '1' bits in a binary word. For an 8-bit input, popcount returns a value from 0 (all zeros) to 8 (all ones).

The naive approach of iterating through each bit takes O(n) time. Using an adder tree structure, popcount can be computed in O(log n) time. Full adders (3:2 compressors) are used to sum groups of bits efficiently.

Popcount is a fundamental operation in modern processors, supported by dedicated instructions like x86's POPCNT and ARM's VCNT. It's essential for cryptography (checking parity), error correction (Hamming codes), and database operations (bitmap indexes).

Architecture

📐 8-bit Popcount Tree

Input: b7  b6  b5  b4  b3  b2  b1  b0

Level 1: Count pairs (half adders)
        b7+b6   b5+b4   b3+b2   b1+b0
          │       │       │       │
        [2b]    [2b]    [2b]    [2b]   ← Each outputs 0-2

Level 2: Merge pairs (2-bit adders)
        [2b]+[2b]       [2b]+[2b]
            │               │
          [3b]           [3b]          ← Each outputs 0-4

Level 3: Final merge (3-bit adder)
              [3b]+[3b]
                  │
                [4b]                   ← Outputs 0-8

Total: log₂(8) = 3 levels
Building Blocks
Half Adder: 2-bit → 2-bit count (0-2)
sum = a ⊕ b
carry = a · b

Full Adder: 3-bit → 2-bit count (0-3)
sum = a ⊕ b ⊕ c
carry = ab + bc + ac

Worked Example

8-bit Popcount: Input = 11011011

Input:    1 1 0 1 1 0 1 1  (decimal 219)

Level 1 - Pair sums (half adders):
  Pair 3: 1+1 = 10 (2)    Pair 2: 0+1 = 01 (1)
  Pair 1: 1+0 = 01 (1)    Pair 0: 1+1 = 10 (2)

Level 2 - Merge pairs (2-bit adders):
  Left:  10 + 01 = 011 (3)
  Right: 01 + 10 = 011 (3)

Level 3 - Final sum (3-bit adder):
  011 + 011 = 0110 (6)

Result: popcount(11011011) = 6  ✓

SWAR (SIMD Within A Register) Method

// Parallel popcount using arithmetic

Input x = 0xB4 = 1011 0100

Step 1: Count bits in pairs
  x = (x & 0x55) + ((x >> 1) & 0x55)
  x = (10110100 & 01010101) + ((01011010) & 01010101)
  x = 00010100 + 01010000 = 01100100  
  // Each 2-bit field holds count for that pair
  // 01=1, 10=2, 01=1, 00=0

Step 2: Count bits in nibbles
  x = (x & 0x33) + ((x >> 2) & 0x33)
  x = (01100100 & 00110011) + ((00011001) & 00110011)
  x = 00100000 + 00010001 = 00110001
  // Each 4-bit field: 0011=3, 0001=1

Step 3: Count bits in byte
  x = (x & 0x0F) + ((x >> 4) & 0x0F)
  x = (00110001 & 00001111) + ((00000011) & 00001111)
  x = 00000001 + 00000011 = 00000100

Result: popcount(0xB4) = 4  ✓

RTL Implementation

VERILOG popcount.v 
// 8-bit Popcount - Tree Structure
module popcount_8 (
    input      [7:0] data_in,
    output     [3:0] count     // 0-8
);

// Level 1: Count pairs (4 half adders)
wire [1:0] p3 = data_in[7] + data_in[6];
wire [1:0] p2 = data_in[5] + data_in[4];
wire [1:0] p1 = data_in[3] + data_in[2];
wire [1:0] p0 = data_in[1] + data_in[0];

// Level 2: Merge pairs (2 adders)
wire [2:0] q1 = p3 + p2;  // 0-4
wire [2:0] q0 = p1 + p0;  // 0-4

// Level 3: Final sum
assign count = q1 + q0;   // 0-8

endmodule

// 32-bit Popcount - Behavioral
module popcount_32 (
    input      [31:0] data_in,
    output reg [5:0]  count    // 0-32
);

integer i;
always @(*) begin
    count = 0;
    for (i = 0; i < 32; i = i + 1)
        count = count + data_in[i];
end

endmodule

// 32-bit Popcount - SWAR Method
module popcount_32_swar (
    input      [31:0] data_in,
    output     [5:0]  count
);

// Constants for SWAR
localparam [31:0] M1 = 32'h55555555;  // 0101...
localparam [31:0] M2 = 32'h33333333;  // 0011...
localparam [31:0] M4 = 32'h0F0F0F0F;  // 00001111...

// Step 1: Count bits in pairs
wire [31:0] step1 = (data_in & M1) + ((data_in >> 1) & M1);

// Step 2: Count bits in nibbles
wire [31:0] step2 = (step1 & M2) + ((step1 >> 2) & M2);

// Step 3: Count bits in bytes
wire [31:0] step3 = (step2 & M4) + ((step2 >> 4) & M4);

// Step 4: Sum bytes (multiply by 0x01010101 and shift)
wire [31:0] step4 = step3 * 32'h01010101;

assign count = step4[31:26];

endmodule

// 64-bit Popcount - Tree
module popcount_64 (
    input      [63:0] data_in,
    output     [6:0]  count    // 0-64
);

wire [5:0] cnt_hi, cnt_lo;

popcount_32 u_hi (.data_in(data_in[63:32]), .count(cnt_hi));
popcount_32 u_lo (.data_in(data_in[31:0]),  .count(cnt_lo));

assign count = cnt_hi + cnt_lo;

endmodule
VHDL popcount.vhd 
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;
use IEEE.NUMERIC_STD.ALL;

entity popcount_8 is
    port (
        data_in : in  std_logic_vector(7 downto 0);
        count   : out std_logic_vector(3 downto 0)
    );
end popcount_8;

architecture tree of popcount_8 is
    -- Level 1: 2-bit sums
    signal p3, p2, p1, p0 : unsigned(1 downto 0);
    -- Level 2: 3-bit sums
    signal q1, q0 : unsigned(2 downto 0);
begin
    -- Level 1: Pair counts
    p3 <= ("0" & data_in(7)) + ("0" & data_in(6));
    p2 <= ("0" & data_in(5)) + ("0" & data_in(4));
    p1 <= ("0" & data_in(3)) + ("0" & data_in(2));
    p0 <= ("0" & data_in(1)) + ("0" & data_in(0));
    
    -- Level 2: Merge pairs
    q1 <= ("0" & p3) + ("0" & p2);
    q0 <= ("0" & p1) + ("0" & p0);
    
    -- Level 3: Final sum
    count <= std_logic_vector(("0" & q1) + ("0" & q0));
end tree;

-- Generic popcount
entity popcount is
    generic (WIDTH : natural := 32);
    port (
        data_in : in  std_logic_vector(WIDTH-1 downto 0);
        count   : out natural range 0 to WIDTH
    );
end popcount;

architecture behavioral of popcount is
begin
    process(data_in)
        variable cnt : natural range 0 to WIDTH;
    begin
        cnt := 0;
        for i in 0 to WIDTH-1 loop
            if data_in(i) = '1' then
                cnt := cnt + 1;
            end if;
        end loop;
        count <= cnt;
    end process;
end behavioral;

Interview Notes

  • Popcount (Hamming weight) counts the number of 1 bits. Output range: 0 to n.
  • Tree structure: log₂(n) levels. 8-bit needs 3 levels, 32-bit needs 5 levels.
  • Basic building blocks: half adders (2→2) and full adders (3→2).
  • SWAR trick: x = (x & 0x55) + ((x >> 1) & 0x55) counts pairs in parallel.
  • Parity = popcount mod 2 = XOR of all bits (much simpler circuit).
  • Hardware: x86 POPCNT instruction, ARM VCNT, RISC-V Zbb extension.
  • Common application: Hamming distance = popcount(a XOR b).
