Truth Table

#

The half subtractor computes A - B. When B > A (subtracting 1 from 0), we need to borrow from a higher bit position.

A B Difference Borrow Explanation
00000 - 0 = 0
01110 - 1 = -1 (borrow)
10101 - 0 = 1
11001 - 1 = 0

Borrow semantics: The borrow output indicates that we need to "borrow 1" from the next higher bit position. In binary: 0 - 1 = (10 - 1) = 1 with borrow.

Boolean Expression

#

Difference

D = A ⊕ B

Borrow

Borrow = A' · B

Canonical SOP (minterms)

D = Σm(1, 2)  |  Borrow = Σm(1)

Key insight: The Difference output is identical to the Half Adder's Sum (XOR). Only the Borrow differs — it's A'·B instead of A·B (the carry).

Derivation

#

From Truth Table to Boolean Expression

Difference (D): Output is 1 when inputs differ

D = A'B + AB' = A ⊕ B

Borrow: Output is 1 only when A=0 and B=1

Borrow = A'B

Comparison with Half Adder

Half Subtractor and Half Adder share the XOR gate for their primary output. The difference is in the secondary output:

Half Adder: Carry = A · B
Half Sub: Borrow = A' · B
Converting HA to HS

To convert a Half Adder to Half Subtractor, simply invert the minuend (A) input before the AND gate that generates the carry/borrow.

Borrow = NOT(A) · B

The half subtractor requires only three gates:

  • XOR gate: Computes Difference = A ⊕ B
  • NOT gate: Inverts A to produce A'
  • AND gate: Computes Borrow = A' · B

RTL Implementation

#
Verilog half_subtractor.v 
// Half Subtractor — Dataflow style
module half_subtractor (
    input  wire a,      // Minuend
    input  wire b,      // Subtrahend
    output wire diff,   // Difference
    output wire borrow  // Borrow output
);

    assign diff   = a ^ b;      // XOR for difference
    assign borrow = ~a & b;     // A' AND B

endmodule


// ─── Gate-level implementation ─────────────────────
module half_subtractor_gate (
    input  a, b,
    output diff, borrow
);

    wire a_not;

    not U1 (a_not, a);
    xor U2 (diff, a, b);
    and U3 (borrow, a_not, b);

endmodule


// ─── Testbench ─────────────────────────────────────
module tb_half_subtractor;
    reg a, b;
    wire diff, borrow;

    half_subtractor dut (.a(a), .b(b), .diff(diff), .borrow(borrow));

    initial begin
        $display("A B | Diff Borrow");
        $display("----+-----------");
        for (integer i = 0; i < 4; i = i + 1) begin
            {a, b} = i[1:0];
            #10;
            $display("%b %b |  %b     %b", a, b, diff, borrow);
        end
        $finish;
    end
endmodule
VHDL half_subtractor.vhd 
-- Half Subtractor - Dataflow architecture
library IEEE;
use IEEE.STD_LOGIC_1164.all;

entity half_subtractor is
    port (
        a      : in  std_logic;  -- Minuend
        b      : in  std_logic;  -- Subtrahend
        diff   : out std_logic;  -- Difference
        borrow : out std_logic   -- Borrow output
    );
end half_subtractor;

architecture dataflow of half_subtractor is
begin
    diff   <= a xor b;
    borrow <= (not a) and b;
end dataflow;

Applications

#
Building Full Subtractors

Two half subtractors plus an OR gate form a full subtractor, just as half adders build full adders.

Comparison Circuits

The borrow output can indicate A < B, useful in magnitude comparators and sorting networks.

Down Counters

Half subtractors can implement the LSB stage of binary down counters when counting by 1.

Educational Tool

Demonstrates binary subtraction fundamentals and the concept of borrowing in digital systems.

Interview Notes

#
  • What's the difference between half subtractor and half adder? — Same Difference/Sum (XOR), but Borrow = A'·B while Carry = A·B. The NOT is on different inputs.
  • Why is it called "half"? — It cannot accept a borrow-in from a previous stage, so it can only handle the LSB of a multi-bit subtraction.
  • How does borrow work? — When subtracting 1 from 0, we "borrow" 2 from the next column: (2-1=1), and the borrow propagates.
  • Gate count? — Minimum 3 gates: 1 XOR, 1 NOT, 1 AND. Can also use 1 XOR + 1 AND-OR-INVERT.
  • How to build from NAND? — Requires 5 NAND gates: 4 for XOR, 1 for Borrow (with input inversion).
  • Relation to two's complement? — Subtraction A-B can be done as A + (~B) + 1 using adders. Half subtractor is the direct approach.
