Boolean Expression

#

Sum

Sum = A ⊕ B ⊕ Cin

Carry-out

Cout = (A · B) + (Cin · (A ⊕ B))

Canonical SOP — Sum (minterms)

Sum = Σm(1, 2, 4, 7)  |  Cout = Σm(3, 5, 6, 7)

Key insight: The Sum output is a 3-input XOR — odd parity of inputs. The carry-out is the majority function: 1 when two or more inputs are 1.

Circuit Diagram

#

A full adder can be constructed from two half adders and an OR gate:

  • Half Adder 1: Adds inputs A and B
  • Half Adder 2: Adds the sum from HA1 with Cin
  • OR gate: Combines the carries from both half adders

RTL Implementation

#
Verilog full_adder.v 
// Full Adder — gate-level / dataflow
module full_adder (
    input  wire a,
    input  wire b,
    input  wire cin,
    output wire sum,
    output wire cout
);
    // Dataflow style
    assign sum  = a ^ b ^ cin;
    assign cout = (a & b) | (cin & (a ^ b));

endmodule


// ─── Structural: two Half Adders ───────────────────
module half_adder (input a, b, output sum, cout);
    assign sum  = a ^ b;
    assign cout = a & b;
endmodule

module full_adder_struct (
    input  a, b, cin,
    output sum, cout
);
    wire s1, c1, c2;
    half_adder HA1 (.a(a),  .b(b),  .sum(s1), .cout(c1));
    half_adder HA2 (.a(s1), .b(cin), .sum(sum), .cout(c2));
    assign cout = c1 | c2;
endmodule

Timing Behavior

#

The critical path runs from any input through the XOR chain to Sum (2 XOR gate delays) and through the AND-OR to Cout (typically 2–3 gate delays depending on implementation).

In a ripple-carry adder, the Cout propagation delay accumulates across N stages, making carry the timing bottleneck. This motivates carry-lookahead and prefix adder architectures.

Hazard note: XOR-based Sum path can exhibit dynamic hazards. Insert glitch filters when synthesizing for asynchronous downstream logic.

Interview Notes

#
  • What is the critical path? — The path through the carry chain. Sum has 2 XOR delays, Cout has 2-3 gate delays.
  • How does ripple-carry delay scale? — O(n) for n bits. Carry-lookahead achieves O(log n).
  • Can you implement using only NAND gates? — Yes, requires 9 NAND gates for a minimal implementation.
  • Derive from two half adders? — HA1 adds A+B, HA2 adds result+Cin, OR combines carries.
  • What are G and P signals? — Generate (G=AB) and Propagate (P=A⊕B) enable carry-lookahead optimization.
  • FPGA implementation? — FPGAs use dedicated carry chains that bypass LUT logic for faster ripple.
